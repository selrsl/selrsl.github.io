Midterm Content

Assignments

Assignment #1 - Data Science

This assignment focused on data science, where we defined it, compared it to business intelligence, and discussed the role of data visualization. I highlighted how data science uses techniques like machine learning to predict outcomes, going beyond business intelligence. I also covered how algorithms, machine learning, and programming are essential to data science. It made me realize that data science is about transforming data into actionable insights.

Activities

About Me

In this activity, we introduced ourselves using a template with details like our name, three words to describe ourselves, and our goals for the course. It helped us reflect on our academic journey and connect with classmates.



Activity #1 – Leveraging Big Data and Data Science

This group activity explored how RetailCo, a retail company, could use big data and data science to improve operations. I researched data types like customer transactions and social media activity, showing how each could inform decisions like pricing, promotions, and inventory management. It deepened my understanding of how data drives business success.



Activity #2 – Operators

This activity explores operators in R programming, focusing on using concatenation with the `c()` function to combine data types like numbers and strings. It enhances understanding of data manipulation in R. Reflecting on this, concatenation proves to be a powerful tool for organizing and analyzing data.



Activity #3 – Working with Vectors and Data Frames

This activity involves working with vectors and data frames in R. Tasks included creating a numeric vector `x`, extracting even numbers to calculate their sum, constructing an `employees` data frame, and writing a function to extract unique values from a column. I also checked for salaries above 70,000, set a salary value to `NA`, and calculated the total salary excluding `NA`. This enhanced my understanding of data manipulation and handling missing values in R.



Activity #4 – Manipulating and Merging Data

This activity focuses on manipulating and combining datasets using R. First, I created two data frames, `df1` and `df2`, and performed an inner join on the `ID` column. I then added a new column that categorizes `Age` into "Young", "Middle-aged", or "Senior" based on age ranges. Using the merged data, I computed the average score for each age group. Next, I sorted the data by `Subject` in ascending order and `Score` in descending order. Lastly, I wrote an R function that combines multiple data frames using `rbind()` after ensuring the columns match. This helped me better understand data merging, grouping, sorting, and combining in R.



Activity #5 – Using Built-in Functions and Control Structures

This activity involves using R's built-in functions and control structures. First, I worked with a vector of names and converted all names to title case, then added the prefix "ID_" to each name. I also created an R function to calculate the mean, median, and standard deviation of a numeric vector, testing it with a random vector. I then wrote a script to classify each number in a vector as "Positive", "Negative", or "Zero", handling edge cases like empty vectors. For a vector of scores, I calculated the average score for values above 80. Lastly, I wrote a recursive function to compute the nth Fibonacci number. This exercise helped deepen my understanding of control structures, functions, and handling edge cases in R.



Activity #6 – Using APPLY Family FUNCTIONS

This activity involved using `apply()`, `lapply()`, and `sapply()` functions to analyze students' grades. I calculated each student's average score with `apply()` and added it as a new column. Then, I used `lapply()` to find the mean score for each subject and `sapply()` for the total score and standard deviation. I also identified the highest score in each subject and labeled students as "Pass" or "Fail" based on their average score using `apply()`. This task improved my skills in applying R functions for data manipulation and analysis.
